# Smart Commute Predictor

AI-powered public transport prediction system built using React, FastAPI, Scikit-Learn, PostgreSQL and OpenStreetMap.

## Features

* ETA Prediction
* Seat Availability Prediction
* Live Bus Tracking
* Crowd Level Monitoring
* Analytics Dashboard
* Route Performance Analysis

## Tech Stack

Frontend:

* React
* Vite
* Leaflet
* Recharts

Backend:

* FastAPI
* Python
* Scikit-Learn

Database:

* PostgreSQL
